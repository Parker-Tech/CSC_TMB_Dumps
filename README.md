# CSC_TMB_Dumps
TMB dumps related to MiniCSC Ecogas Longevity Project

When saving TMB dumps during data acquisition runs (DAQ) save in this format:
```bash
.
└── DATE_HV_/
    └── SRC_HOLE_RUN#

example with source:
.
└── 241025_HV3600/
    ├── Cd_hole4_run00
    ├── Cd_hole4_run01
    ├── Cd_hole4_run02
    ├── dark_run00
    ├── dark_run01
    └── dark_run02
    
```
