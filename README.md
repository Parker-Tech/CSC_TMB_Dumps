# CSC_TMB_Dumps
##### TMB dumps related to MiniCSC Ecogas Longevity Project

When saving TMB dumps during data acquisition runs (DAQ) save in this format:
```bash
.
└── DATE_HV_/
    └── RUN_SRC_HOLE

example with source:
.
└── 241025_HV3600/
    └── run00_Cd_hole4

example dark run:
.
└── 241025_HV3600/
    └── run00_dark
```
